What Is FTEdit ?

FTedit is an open-source software of a Fault Tree editor. A Fault Tree is a representation of a technical hazard of a complex system.

With FTEdit you can create , edit , save , load , export and analyse all types of fault tree !

How to use FTEdit ?

In FTEdit you can create and edit a fault tree , adding event , gate and probabilities. When your fault tree is set and done you can analyze it buy clicking on the button.

Creation and Editing of a Fault Tree

In a Fault Tree there's several type of Gate : AND , OR , XOR , INHIBIT , and VotingOR , you can select them directly in the tool bar.

You can also link up two fault tree by using a transfer gate

Saving and Loading of a Fault Tree

To save a fault tree, it has to be correct, the software will automatically check if it does, if it is correct click on the button to save your fault tree

If you want to load or open an existing project of a fault tree your file must in .opsa or xml format. To load a project click on the and select your fault tree !

Analyze a Fault Tree

Once your fault tree is done and you want to get some numbers about it, the analyzer will check if the logic of the tree is valid.If not it will return errors messages to fix on the fault tree.

To lunch fault tree computing, click on the button and choose your type of evaluation. You can choose both quantitative and qualitative computing. Result will be displayed in the explorer of the application (at the left bar)
